How it works

Event-bounded, venue-level presence.

  1. ✓ Create a Loop with start/end time and invite friends or colleagues.
  2. ✓ Add Pins (houses, offices, booths); geofencing auto check-in/out.
  3. ✓ See Live Presence like "Anya @ Main Hall • 12 min".
  4. ✓ Use Visit Coach: progress ring, upcoming stops, reciprocity nudges.
  5. ✓ Loop expires → data auto-wipes. Optional private recap.
